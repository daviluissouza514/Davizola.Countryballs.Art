# Como Abrir

Você pode abrir dos seguintes jeitos:

1. Vai para a Página de [Downloads](https://scratch.mit.edu/download) do Scratch

2. Escolha o seu sistema (pode ser Windows, MacOS, ChromeOS e Android)

3. Clique no Download direto (ou o download na loja do sistema)

4. Depois de baixar, instale e aproveite meus projetos

OBS: O download do Windows direto da página do Scratch pode ficar indisponível, por isso baixe na Microsoft Store

Segundo jeito:

1. Vai para https://desktop.turbowarp.org

2. Escolha seu sistema e clique em download (64 ou 32 bits, ou ARM64)

3. Instale e aproveite

Terceiro jeito:

1. Acesse o site do [Scratch](https://scratch.mit.edu/projects/editor)

2. Clique em Criar (ou abre um projeto existente)

3. Vai em cima e clique em: Arquivo > Load from your computer

4. Ele vai abrir uma janela pedindo o .sb3

5. Selecione o .sb3 que você baixou daqui e pronto! (só não o recompartilhe, ok?)

Só isso mesmo, esse foi o readme :)
